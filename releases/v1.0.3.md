## Installation

### Docker Deployment
```bash
git clone https://github.com/Inton1701/POSExpress.git
cd POSExpress
git checkout v1.0.3
docker compose up --build -d
```

```bash
git clone https://github.com/Inton1701/POSExpress.git
cd POSExpress
git checkout v1.0.3
./deploy-all.sh
```
