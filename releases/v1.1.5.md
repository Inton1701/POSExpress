## Installation

### Docker Deployment
```bash
git clone https://github.com/Inton1701/POSExpress.git
cd POSExpress
git checkout v1.1.5
docker compose up --build -d
```

### Native Deployment
```bash
git clone https://github.com/Inton1701/POSExpress.git
cd POSExpress
git checkout v1.1.5
./deploy-all.sh
```

### Update from Existing Installation
```bash
./auto-update.sh
```
