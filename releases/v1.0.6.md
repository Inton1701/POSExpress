## Installation

### Docker Deployment
```bash

cd POSExpress
git checkout v1.0.6
docker compose up --build -d
```

### Native Deployment
```bash
git clone https://github.com/Inton1701/POSExpress.git
cd POSExpress
git checkout v1.0.6
./deploy-all.sh
```

```bash

```
